SmartLender

Next-Generation Loan Approval System

About the Project

SmartLender is a machine learning-powered web application designed to streamline the loan approval process. By analyzing historical loan data, our system can accurately predict the likelihood of loan approval based on an applicant's financial and demographic profile.

🤖 XGBoost AI

Powered by an optimized XGBoost classifier for maximum accuracy.

🛡️ Robust Validation

Tuned using RandomizedSearchCV and validated with 5-Fold Cross-Validation.

⚡ Real-Time

Get instant predictions and confidence scores via our Flask backend.
